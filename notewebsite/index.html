Responsive Design


Responsive Design is process of designing websites that is readable and enjoyable on all type of devices regardless of their height and width. A responsive web design will automatically adjust for different screen sizes and viewports. Responsive Design use various css media queries and flexible grid.

HTML


The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. HTML stand for Hyper Text Markup Language. It is the foundation of website and web application development. HTML marks up the content of a site. Basically, it tells the user's computer what things are.

CSS


Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts. CSS id used to take a plain HTML website to the next level.

Bootstrap is a frontend framework using HTML, CSS, jQuery. It is designed to help developer quickly and easily build responsive, mobile-ready websites that are cross-browser compatible. Bootstrap is a giant collection of handy, reusable bits of code written in HTML, CSS, and JavaScript.
