HTML

What is HTML?

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. HTML stand for Hyper Text Markup Language. It is the foundation of website and web application development. HTML marks up the content of a site. Basically, it tells the user's computer what things are.

HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables. It allows you to ensure that your content is understood by both segments of your audience: the people who watch, read, or listen to your content, and the computers that display it. HTML is very easy to learn and use. Once you learn the basic of html you can create your own website within a week of learning HTML.


Example

<!DOCTYPE html>
<html>
<head >
<title>Page Title</title >
</head>
<body >

<h1>This is a Heading</h1>
<p > This is a paragraph.</p>

</body>
</html >

Example Explained

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Basic HTML tags

List of important tags
Tag Description
<html> Defines the root of an HTML document
<head> Contains metadata/information for the document
<body> Defines the document's body
<header> Defines a header for a document or section
<h1> to <h6> Defines HTML headings
<p> Defines a paragraph
<ul> Defines an unordered list
<ol> Defines an ordered list
<li> Defines list item
<img> Defines embedded image content
<video> Defines embedded video content
<audio> Defines embedded audio content
<a> Defines a hyperlink
<div> Defines a section in a document
<span> Defines a section in a document
<form> Defines an HTML form for user input
<table> Defines a table
